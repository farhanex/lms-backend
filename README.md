### Installation

1. **Clone the repository**:
    ```sh
    git clone https://github.com/farhanoxg/lms-backend.git
    ```

2. **Install backend dependencies**:
    ```sh
    cd lms-backend
    npm install
    ```
    ### Running the Application

1. **Start  backend server**:
    ```sh
    cd lms-backend
    npm start
    ```
